## Philosophy

These guides document real-world practices, not theoretical ideals. They emphasize:

- **Human guidance** - AI assists, humans decide
- **Concrete outcomes** - Testable success criteria
- **Iterative improvement** - Learn and adapt
- **Pattern recognition** - Know what to check
