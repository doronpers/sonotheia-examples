# Scripts Directory

This directory contains utility scripts for the sonotheia-examples repository.

## standardize_repo.sh

Repository standardization script. Applies global standards including:
- Governance documentation deployment
- Code formatting configuration
- Pre-commit hooks setup

Usage: `./scripts/standardize_repo.sh`

## Launcher Scripts

- **launcher.sh** - Main launcher for the repository
- **quicktest.sh** - Quick test runner

## Example Scripts

Example-specific scripts are located in their respective directories:
- `examples/curl/*.sh` - cURL example scripts
- `examples/terraform/aws/lambda/build_lambda.sh` - Lambda build script
